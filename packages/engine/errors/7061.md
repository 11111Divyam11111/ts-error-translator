Are you accidentally creating a mapped type with an interface?

```ts
interface PascalName {
  FirstName: string;
  LastName: string;
}

type CamelCaseNameKeys = Uncapitalize<keyof PascalName>;

interface CamelCaseName {
  [P in CamelCaseNameKeys]: string;
}
// 👎
```
